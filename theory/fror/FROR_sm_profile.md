## 0. Статус

- Слой: `Profiles`.
- Статус: `Conjectural`.
- Назначение: зафиксировать условия, при которых SM-like структура может
  быть интерпретирована как эффективный слой FROR.

------------------------------------------------------------------------

## 1. Уровень A (структурный)

### 1.1 Минимальные структурные объекты

- пространство альтернатив `E0` (неоплаченные переходы),
- скаляр `tau` (оплаченные акты фиксации),
- локальные устойчивые возбуждения (SLOT-like объекты),
- coarse-graining с сохраняемыми инвариантами представления.

### 1.2 Базовая гипотеза H1

Существует класс coarse-graining операторов, для которого наблюдаемые
величины определяются как инварианты относительно переописания внутренних
степеней свободы.

Интерпретация: это кандидат на gauge-like структуру представления.

### 1.3 Базовая гипотеза H2

Существуют устойчивые классы возбуждений (мультиплетоподобные структуры),
чьи эффективные параметры (масса/стабильность/взаимодействие) определяются
стоимостью фиксации и топологией альтернатив.

------------------------------------------------------------------------

## 2. Уровень B (интерпретационный)

Допустимый mapping-кандидат:
- gauge-like инвариантности -> кандидаты на `U(1)`, `SU(2)`, `SU(3)`;
- классы возбуждений -> particle-like мультиплеты;
- стоимость фиксации/стабилизации -> effective mass hierarchy.

Это изоморфизм-кандидат, а не тождество со Стандартной моделью.

------------------------------------------------------------------------

## 3. Явные ограничения профиля

Профиль не утверждает:
1. что `U(1) x SU(2) x SU(3)` уже строго выведены из канона FROR;
2. что механизм Хиггса/поколений/конфайнмента уже формально доказан;
3. что FROR заменяет Стандартную модель как эмпирическую теорию.

Корректная формулировка:
- FROR задает мета-структурные условия, при которых SM-like описание
  рассматривается как правдоподобный эффективный кандидат.

------------------------------------------------------------------------

## 4. Условия применимости

Профиль применим, если явно заданы:
1. эквивалентностные классы переописаний (gauge-like действия);
2. наблюдаемые инварианты, устойчивые к этим переописаниям;
3. правила формирования и устойчивости SLOT-like возбуждений;
4. протокол калибровки между операционными параметрами и физическими шкалами.

Без этого SM-mapping остается качественной гипотезой.

------------------------------------------------------------------------

## 5. Фальсифицируемые тесты профиля

Профильная программа считается опровергнутой для выбранной нормализации,
если воспроизводимо наблюдается одно из:
1. отсутствие нетривиальных инвариантов представления при заявленном
   gauge-like действии;
2. отсутствие устойчивых классов возбуждений при увеличении масштаба;
3. невозможность воспроизвести хотя бы качественную иерархию effective-масс
   из стоимости фиксации/стабилизации;
4. сильная зависимость результатов от частных трюков параметризации
   (без устойчивого ядра инвариантов).

------------------------------------------------------------------------

## 6. Минимальная экспериментальная программа

1. Построить семейства coarse-graining с проверкой инвариантов представления.
2. Идентифицировать устойчивые классы возбуждений и их мультиплетную структуру.
3. Оценить связь "стоимость фиксации -> effective mass".
4. Проверить воспроизводимость результатов на разных классах микродинамики.

------------------------------------------------------------------------

## 7. Рабочая формула статуса

SM Profile в FROR:
- не "вывод Стандартной модели",
- а программа поиска структурных условий, при которых SM-like описание
  возникает как устойчивый эффективный слой.

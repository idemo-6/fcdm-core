## 1. Базовые определения (уровень A)

- Система: пространство траекторий фиксированной микродинамики.
- Наблюдатель: `O = (M_O, Sigma_O, pi_O)`.
- `M_O`: конечная внутренняя память (в битах).
- `pi_O`: стратегия coarse-graining траекторий в классы `P = {P_1, ..., P_K}`.
- Симметрия классов: перестановочная инвариантность классов при фиксированных `O, M_O, pi_O`.
- `DeltaS_defect`: изменение дефектной энтропии на актах фиксации.

Под "допустимой стратегией" понимается функция, вычислимая конечным
детерминированным автоматом с памятью `M_O`, без внешнего оракула, внешней
памяти и внешней случайности.

------------------------------------------------------------------------

## 2. Нетривиальность режима различения

В этом документе рассматривается не любая бинарная детекция, а
структурно-нетривиальный режим, совместимый с FROR-трихотомией:

`D = {-, 0, +}`.

Интерпретация:
- `0`: нейтральный/флуктуационный класс (без направленного вклада),
- `+`: направленный класс усиления,
- `-`: направленный класс релаксации.

Бинарное разбиение `0/1` допускается как деградационный режим детекции факта,
но не считается достаточным для нетривиальной направленной динамики сложности.

------------------------------------------------------------------------

## 3. Лемма о недостаточности 1 бита (для трихотомии)

При `M_O = 1` наблюдатель не может устойчиво и воспроизводимо реализовать
трехклассовое разбиение `{- ,0,+}` без склеивания классов.

Следствие: для структурно-нетривиального нарушения симметрии в режиме
трихотомии требуется минимум `M_O >= 2` бита.

------------------------------------------------------------------------

## 4. Теорема о пороге памяти и росте дефектной энтропии

### Теорема

Пусть:
1. используется трихотомный режим `{- ,0,+}`;
2. coarse-graining неинъективен (реальное склеивание альтернатив);
3. фиксация результата необратима и имеет положительную цену (Landauer-тип);
4. отсутствуют внешний оракул/внешняя память.

Тогда:
- при `M_O >= 2` существует допустимая стратегия, нарушающая симметрию классов;
- на актах платной фиксации наблюдается строгий рост `DeltaS_defect > 0`;
- при `M_O = 1` в этом режиме нарушение симметрии и устойчивый рост
  `DeltaS_defect` недостижимы.

### Важная оговорка

Утверждение про `M_O = 1` относится именно к нетривиальной трихотомии.
Для деградационного бинарного режима возможна простая детекция, но она не
считается здесь источником направленной динамики сложности.

------------------------------------------------------------------------

## 5. Первый рабочий оператор (M_O = 2)

Пример стратегии: "инверсия по пересечению".

- фиксируется класс "оба `dir != 0`";
- случаи "хотя бы один `dir = 0`" трактуются как флуктуационные;
- платная фиксация учитывается отдельно.

Корректная интерпретация результата:
- `DeltaS_defect > 0` на актах платной фиксации;
- `DeltaS_defect = 0` на шагах без фиксации.

------------------------------------------------------------------------

## 6. Экспериментальный протокол

### 6.1 Цель

Проверить:
1. порог `M_O >= 2` для трихотомного режима;
2. строгий рост `DeltaS_defect` именно на платных актах фиксации.

### 6.2 Базовая серия (M_O = 2)

Вход:
- ансамбль траекторных объектов (или эквивалентных тест-кейсов);
- метки `dir in {-1,0,+1}`;
- стратегия "инверсия по пересечению".

Выход (CSV):
- `N, term_id, step, dir1, dir2, intersection_flag, paid_fixation, DeltaS_before, DeltaS_after, tau`

Критерии:
- среднее `DeltaS_after - DeltaS_before > 0` на `paid_fixation = 1`;
- среднее `DeltaS_after - DeltaS_before = 0` на `paid_fixation = 0`;
- статистическая значимость (например, `p < 0.001`).

### 6.3 Серии M_O = 3,4,5,...

- повтор базовой процедуры;
- кривая эффекта `E(M_O) = <DeltaS>` на платных актах;
- анализ насыщения/стабилизации.

### 6.4 Контроли

- инвариантная стратегия без неинъективного склеивания;
- стратегия без платной фиксации;
- контроль на артефакты реализации.

Если контроль дает систематический рост `DeltaS_defect`, реализация некорректна.

------------------------------------------------------------------------

## 7. Критерий фальсификации

Связка "порог памяти + рост дефектной энтропии" считается опровергнутой для
данной нормализации, если наблюдается хотя бы одно:

1. устойчивое трехклассовое различение при `M_O = 1` без внешнего оракула;
2. отсутствие роста `DeltaS_defect` на платных фиксациях при `M_O >= 2`;
3. систематический рост `DeltaS_defect` в контрольных режимах, где его быть
   не должно.

------------------------------------------------------------------------

## 8. Короткая формулировка

Симметрия в нетривиальном трихотомном режиме `{- ,0,+}` не нарушается без
минимального ресурса памяти: первый устойчивый порог — `M_O = 2` бита.
Рост дефектной энтропии возникает не из самой классификации, а из
неинъективного coarse-graining с платной необратимой фиксацией.

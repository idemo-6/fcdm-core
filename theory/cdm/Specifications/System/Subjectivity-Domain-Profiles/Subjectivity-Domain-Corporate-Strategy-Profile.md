# Profile: Subjectivity in Corporate Strategy

Status: `illustrative`

## 1. Контекст

- Область: корпоративное управление.
- Цель: удержание ключевых специалистов/устойчивость стратегии под ограничениями.
- Критерий `R`: доменная метрика результата (например, retention rate).

## 2. Reference

Эталон `Ref` задается по
[[fcdm-core/theory/cdm/Specifications/System/Subjectivity-Reference-Agent-Profile|Subjectivity-Reference-Agent-Profile]].

## 3. Rule

- Высокий `S` не считается риском сам по себе.
- Оценка решения выполняется только в паре `R + S`.
- Целевой режим: `R >= R(Ref)` при повышенном `S`.

## 4. Practical use

Применяется для сравнения стратегий команды/подразделений и выбора
нестандартных, но результативных траекторий изменения.

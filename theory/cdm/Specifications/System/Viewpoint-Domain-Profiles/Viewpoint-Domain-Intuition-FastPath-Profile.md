# Profile: Intuition as Fast-Path

Интуиция рассматривается как ускоренный режим выбора кэшированных композиций операторов
между аналитическим циклом и рефлекторным режимом.

## Rules

1. Fast-path допустим только при достаточном качестве релевантного опыта.
2. Fast-path не отменяет `evaluate`; кэш должен подтверждаться или ослабляться.
3. При росте `Doubt` выше доменного порога fast-path обязан эскалировать в полный цикл или MetaChangeFlow.

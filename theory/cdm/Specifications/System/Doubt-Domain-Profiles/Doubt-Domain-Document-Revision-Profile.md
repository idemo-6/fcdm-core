# Profile: Doubt in Document Revision

Профиль задает использование `Doubt` при выборе траектории редакторского ChangeFlow.

## 1. Минимум профиля

- список альтернатив ревизии (структура/термины/доказательства);
- метрика операторной готовности редакторского контура;
- пороги для дополнительного review/meta-flow;
- связь с `EntityId@VersionState` документа.

## 2. Назначение

Снижение риска неустойчивых правок и повышение обоснованности перехода в `implement`.

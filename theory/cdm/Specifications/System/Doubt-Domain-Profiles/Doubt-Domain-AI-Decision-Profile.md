# Profile: Doubt in AI Decision

Профиль задает использование `Doubt` для эскалации decision-процедур в ИИ-контурах.

## 1. Минимум профиля

- правила оценки альтернатив `v_j`;
- правила расчета операторной емкости `T`;
- пороги `theta_warning/theta_meta`;
- обязательные evidence для перехода в `implement`.

## 2. Назначение

Снижение риска ложной уверенности при выборе траектории действия.

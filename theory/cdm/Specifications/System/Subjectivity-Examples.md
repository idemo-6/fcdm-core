# Примеры: Subjectivity

## 1. Числовой пример

Вход:

- `i1: W=3, R=1.0, D=0.80, Q_A=0.90, Q_B=0.85`
- `i2: W=1, R=1.0, D=0.50, Q_A=0.60, Q_B=0.70`
- `i3: W=0.5, R=0.5, D=0.20, Q_A=0.95, Q_B=0.90`

Сумма вкладов: `~4.385`.

При `f(x)=ln(1+x)`:

`S ~ 1.700`.

---

## 2. Примеры интерпретации

1. Идентичные системы в том же `C` -> `S=0`.
2. Разные инструменты при высоких `W` и `R` -> рост `S`.
3. Смена контекста `C` при тех же системах -> изменение `S`.

---

## 3. Анти-примеры

1. `Q=0` без `Q_min` -> вычисление невалидно.
2. Сравнение без явного контекста `C` -> неканонично.
3. Интерпретация `S` как эмоций/сознания -> неканонично.

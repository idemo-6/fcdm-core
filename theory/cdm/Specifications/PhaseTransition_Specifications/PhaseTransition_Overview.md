# PhaseTransition — общее понятие перехода между фазами

Различаются два типа фазовых переходов:

1. LC-PhaseTransition — переход между фазами LifeCycle (смена режима существования системы)
2. CF-PhaseTransition — переход между фазами ChangeFlow (процесс передачи и согласования результатов)

Оба типа используют канонический оператор:

⇒

# Примеры: ChangeOperators (прикладные кейсы)

## 1. Назначение

Документ содержит иллюстрации и не является нормативным каноном.

---

## 2. Пример: информационная система (SQL-профиль)

1. `CF1`: планирование материализации (`fn.plan_*`).
2. `pt @->??`: внешняя материализация данных (операторы доступа к БД).
3. `CF2`: анализ снимка без прямого I/O.
4. `CF5`: фиксация обновления состояния.
5. `CF6`: запись трассы операторов и результата.

---

## 3. Пример: биологическая система (абстрактный)

1. `CF1`: сенсорный захват сигналов.
2. `CF2`: локальная обработка.
3. `CF3`: прогноз реакции.
4. `CF4`: выбор реакции.
5. `CF5`: реализация моторного/физиологического акта.
6. `CF6`: обновление адаптивного следа.

---

## 4. Пример классификации доступа

| Оператор | Тип доступа | Комментарий |
|---|---|---|
| локальный валидатор | `L` | выполняется внутри системы |
| inherited policy check | `P` | наследуется от parent-контура |
| sibling data handshake | `S` | доступ через смежный интерфейс |
| external API tool | `E` | внешний поставщик оператора |

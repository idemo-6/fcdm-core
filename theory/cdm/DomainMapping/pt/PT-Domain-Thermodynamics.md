# PhaseTransition в термодинамике

Связанные канонические документы:
- Спецификация PhaseTransition
- [[DomainLexicon]]

## Интерпретация

PhaseTransition соответствует:

- фазовым переходам вещества
- критическим явлениям
- смене режимов устойчивости

## Order Parameter

Примеры:

- плотность
- кристалличность
- магнитизация
- теплоёмкость

## Пример

Жидкость → твёрдое:

Start: жидкое состояние  
In‑Transition: кристаллизация  
End: доминирование твёрдой фазы  

result = true → переход завершён

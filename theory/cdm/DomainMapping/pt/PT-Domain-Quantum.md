# PhaseTransition в квантовой динамике

Связанные канонические документы:
- Спецификация PhaseTransition
- Спецификация LifeCycle-6
- Спецификация ChangeFlow-6
- [[DomainLexicon]]

## Интерпретация

PhaseTransition соответствует процессам:

- декогеренции
- квантового скачка
- измерительного взаимодействия
- потери управляемости состояния

## Order Parameter

Примеры:

- когерентность
- вероятность возбуждённого состояния
- доля населения уровня
- fidelity

## Пример

Квантовый скачок:

Start: когерентная суперпозиция  
In‑Transition: взаимодействие с окружением  
End: коллапс состояния  

result = true → переход завершён

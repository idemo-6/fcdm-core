# PhaseTransition в социо‑экономических системах

Связанные канонические документы:
- Спецификация PhaseTransition
- [[DomainLexicon]]

## Интерпретация

PhaseTransition соответствует:

- кризисам
- рыночным переломам
- социальным изменениям
- tipping points

## Order Parameter

Примеры:

- уровень доверия
- ликвидность
- долговая нагрузка
- безработица

## Пример

Рост → кризис:

Start: экономический рост  
In‑Transition: накопление дисбалансов  
End: рецессия  

result = true → кризис реализовался

# ChangeFlow в вычислениях

Связанные канонические документы:
- Спецификация ChangeFlow-6
- Спецификация LifeCycle-6
- [[DomainLexicon]]

## Нотация

Канонические фазы в этом документе: только `CF1..CF6`.

Если в машинном профиле ICSS-IntentFlow встречаются маркеры `collect/analyze/forecast/decide/implement/evaluate`, они трактуются как синтаксические алиасы представления и не заменяют каноническую фазовую нотацию.

## Интерпретация фаз

- CF1: чтение входных данных
- CF2: обработка и валидация
- CF3: генерация вариантов решений
- CF4: выбор алгоритма
- CF5: выполнение операции (запись состояния)
- CF6: логирование результата

## Пример

Транзакция БД:
- CF1: чтение запроса
- CF2: план выполнения
- CF3: варианты индексов
- CF4: выбор плана
- CF5: commit
- CF6: подтверждение
